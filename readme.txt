Maksukaista payment gateway plugin for Woocommerce. This plugin allows you to accept e-payments through most Finnish banks and credit cards. 

== Description ==

This is a plugin for integrating Maksukaista payment gateway with your Woocommerce store. Accept payments with this plugin, you need to have an active contract with [Maksukaista](https://www.paybyway.com/fi/).

= Supported features =
* Accept payments securely
* Optional separate authorize and capture(settle) for credit card payments 

= Supported payment methods =
Nordea, Osuuspankki, Danskebank, Aktia, Säästöpankki, Paikallisosuuspankki, Handelsbanken, S-Pankki, Visa, Visa Electron, Mastercard, Everyday.fi, Joustoraha.

Notice: Plugin uses libcurl for capture requests.

== Installation ==

1. Upload the downloaded .zip archive through Wordpress plugin management installation.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. From Woocommerce settings menu open 'Payment gateways' tab, and select 'Maksukaista'
4. Fill in private key, sub-merchant id, and order number prefix.  First two can be found from Maksukaista merchant portal.
5. Save settings and make test order and payment to confirm everything works.
